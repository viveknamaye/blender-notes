### Transformations:

Grab - G
Rotate - R
Scale - S

Apply transfomations in particular axis :
x for x-axis
y for y-axis
z for z-axis

Undo Transfomations :

Undo Grab - Alt + G
Undo Rotate - Alt + R
Undo Scale - Alt + S

Shift to Local:

X-axis - X x 2
y-axis - Y x 2
Z-axis - Z x 2

Apply all transformation :
Ctrl + A and select apply all transformations
